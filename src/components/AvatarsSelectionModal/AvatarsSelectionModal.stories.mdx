# AvatarsSelectionModal

AvatarsSelectionModal component is a group of elements that can be used to select one or many avatars from list of available. Avatars can be searched for using text input.

This component supports multi and single select of avatars. If `selectionLimit` is `1`, it would be possible to select new avatar without the need to de-select previous. In multi select mode, old avatar need to be de-selected to select a new one.
