## Colorway

[Colorway](https://github.com/helpscout/colorway) is Help Scout's color management syncing system.

You can install it following the instructions on the linked repository and with it you get a command line utility to get Help Scout colors and find close matches.

Example:

```bash
$ colorway find blue.500

blue.500: #1292EE

$ colorway closest #005500

green.900: #23633e
```

Try it out!
