### Custom Content

The contents of Popover can be customized by using `renderHeader` and/or `renderContent` for rendering.

```jsx
<Popover
  renderHeader={() => <Heading>My Title</Heading>}
  renderContent={() => (
    <div>
      <p>My Content</p>
    </div>
  )}
/>
```
