# SelectDropdown

A SelectDropdown component is an enhanced version of the default HTML `<select>` with a stylized menu UI.

Should be used instead of the deprecated `Select`

<Canvas>
  <Story name="default">
    <SelectDropdown
      disabled={boolean('disabled', false)}
      autoInput={false}
      items={createSpec({
        id: faker.datatype.uuid(),
        value: faker.name.firstName(),
      }).generate(20)}
      state={select(
        'state',
        {
          default: 'default',
          error: 'error',
        },
        'default'
      )}
      dropUp={boolean('dropUp', false)}
      maxHeight={text('maxHeight', '200px')}
      maxWidth={text('maxWidth', '100%')}
      limit={number('limit', 15)}
      width={text('width', '100%')}
      onSelect={action('onSelect')}
    />
  </Story>
</Canvas>
