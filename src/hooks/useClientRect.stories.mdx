# useClientRect

Custom hook that allows you to measure a DOM element, almost completely based on [React's guidelines](http://reactjs.org/docs/hooks-faq.html#how-can-i-measure-a-dom-node).

Returns an array with:

1. The measures as calculated by [getBoundingClientRect](http://developer.mozilla.org/en-US/docs/Web/API/Element/getBoundingClientRect)
2. The DOM element the ref points to
3. The `ref` to pass to the desired element

Example:

```jsx
function MeasureExample() {
  const [rect, el, ref] = useClientRect()

  return (
    <>
      <h1 ref={ref}>Hello, world</h1>
      {rect !== null && (
        <h2>The above header is {Math.round(rect.height)}px tall</h2>
      )}
    </>
  )
}
```
